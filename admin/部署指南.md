# "当才当用"管理后台Demo - 跨平台部署指南

## 📦 方式一：直接复制文件夹（最简单）

### 步骤：

1. **复制整个项目文件夹**
   - 将 `dang-cai-dang-yong/admin` 文件夹复制到U盘或其他存储设备
   - 或者压缩为 `admin.zip` 后传输

2. **在目标电脑上解压**
   - 将项目文件夹解压到任意位置，例如：`D:\projects\admin`

3. **安装Node.js（如果尚未安装）**
   - 访问：https://nodejs.org/
   - 下载并安装 LTS 版本（推荐 18.x 或 20.x）
   - 安装完成后，打开命令行，输入 `node -v` 检查是否安装成功

4. **运行项目**
   - 双击运行 `启动Demo.bat`（Windows）
   - 或在命令行中执行：
     ```bash
     cd D:\projects\admin
     npm install
     npm run dev
     ```

5. **访问系统**
   - 打开浏览器访问：http://localhost:3000
   - 用户名：`admin`
   - 密码：`123456`

## ⚠️ 常见问题

### Q1: 提示 "node 不是内部或外部命令"
**A**: 需要先安装 Node.js，访问 https://nodejs.org/ 下载安装

### Q2: npm install 速度很慢
**A**: 使用国内镜像源：
```bash
npm config set registry https://registry.npmmirror.com
npm install
```

### Q3: 端口3000被占用
**A**: 修改 `vite.config.ts` 中的端口号，或执行：
```bash
npm run dev -- --port 3001
```

### Q4: 首次运行依赖安装失败
**A**: 删除 `node_modules` 文件夹和 `package-lock.json`，重新执行：
```bash
rm -rf node_modules package-lock.json
npm install
```
