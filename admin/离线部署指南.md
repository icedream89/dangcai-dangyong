# 离线部署指南

## 适用场景
客户电脑没有网络连接，无法执行 npm install

## 准备工作（在有网络的电脑上）

### 1. 完整安装项目
```bash
cd admin
npm install
```

### 2. 打包整个项目
将以下文件夹/文件全部打包：
- ✅ admin 文件夹（包含 node_modules）
- ✅ 所有源代码
- ✅ 依赖包（已安装的）

### 3. 压缩大小
- 完整大小：约 300-500MB（包含所有依赖）
- 压缩后：约 100-200MB

## 部署步骤（在目标电脑上）

### Windows
1. 解压项目到任意位置
2. 双击 `启动Demo.bat`
3. 由于已包含 node_modules，会跳过安装步骤
4. 直接启动服务器

### 注意事项
⚠️ 确保目标电脑已安装 Node.js
⚠️ 如果目标电脑系统架构不同，可能需要重新安装依赖

## 验证环境
运行前请确保：
- [ ] Node.js 版本 16+ 已安装
- [ ] 端口 3000 未被占用
- [ ] 防火墙允许 Node.js 访问网络
