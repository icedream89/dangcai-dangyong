# 当才当用 - 管理后台系统 Demo

这是"当才当用"政府服务平台的管理后台演示版本，使用 Vue 3 + Element Plus 构建。

## 功能特性

✅ **完整的UI界面** - 所有核心功能模块的可视化界面
✅ **Mock数据** - 内置模拟数据，无需后端即可演示
✅ **响应式设计** - 适配不同屏幕尺寸
✅ **丰富的交互** - 完整的增删改查操作演示

## 已实现模块

### 1. 系统管理
- ✅ 登录认证（演示账号：admin / 123456）
- ✅ 用户管理
- ✅ 系统配置（参数、字典、菜单、日志）

### 2. 企业管理
- ✅ 企业信息管理（列表、新增、编辑、查看、审核）
- ✅ 员工管理（列表、新增、编辑）
- ✅ 批量导入导出功能演示

### 3. 业务管理
- ✅ 分类管理（树形结构、拖拽）
- ✅ 产品管理（图文展示）
- ✅ 政策管理（富文本、附件）
- ✅ 企业课堂（视频/文档）
- ✅ 求助工单（工单流转）
- ✅ 需求管理（采购需求）

## 快速开始

### 安装依赖

```bash
cd admin
npm install
```

### 运行开发服务器

```bash
npm run dev
```

服务将运行在: http://localhost:3000

### 构建生产版本

```bash
npm run build
```

## 技术栈

- **Vue 3.4** - 渐进式JavaScript框架
- **TypeScript** - 类型安全的JavaScript
- **Vite 5** - 新一代前端构建工具
- **Element Plus 2.4** - Vue 3 组件库
- **Pinia** - Vue 状态管理
- **Vue Router 4** - 路由管理
