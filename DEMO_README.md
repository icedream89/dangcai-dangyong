## 🚀 快速启动

### 一、管理后台Demo

#### 方式1: 命令行启动
```bash
# 1. 进入管理后台目录
cd admin

# 2. 安装依赖
npm install

# 3. 启动开发服务器
npm run dev

# 4. 访问系统
# 浏览器打开：http://localhost:3000
# 用户名：admin
# 密码：123456
```

#### 方式2: 双击启动
- **Windows**: 双击 `admin/启动Demo.bat`
- **Mac/Linux**: 运行 `admin/启动Demo.sh`

### 二、小程序Demo

#### 方式一：使用HBuilderX（推荐）⭐
1. 打开HBuilderX
2. 文件 → 打开目录 → 选择 `miniprogram` 文件夹
3. 运行 → 运行到小程序模拟器 → 微信开发者工具

#### 方式二：命令行
```bash
# 1. 进入小程序目录
cd miniprogram

# 2. 安装依赖
npm install

# 3. 运行开发服务器
npm run dev:mp-weixin

# 4. 使用微信开发者工具打开 dist/dev/mp-weixin 目录
```

## ⚠️ 常见问题

### Q: 提示"node不是内部或外部命令"？
**A**: 需要先安装 Node.js：https://nodejs.org/

### Q: npm install 很慢？
**A**: 使用国内镜像：
```bash
npm config set registry https://registry.npmmirror.com
npm install
```

### Q: 微信开发者工具无法打开？
**A**:
1. 确保已安装微信开发者工具
2. 在HBuilderX中配置工具路径
3. 或使用命令行运行后导入项目

### Q: 端口被占用？
**A**: 修改配置文件中的端口号，或关闭占用端口的程序

### Q: 小程序预览空白？
**A**: 
1. 确认已执行 `npm run dev:mp-weixin`
2. 在微信开发者工具中导入 `dist/dev/mp-weixin` 目录
3. 点击"编译"按钮
