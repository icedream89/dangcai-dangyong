# 小程序Demo演示指南

## 🎯 演示目的
向客户展示"当才当用"平台的微信小程序端功能和界面效果。

## 📱 演示前准备

### 1. 安装必需工具（选择一种）

#### 方案A：HBuilderX（推荐，最简单）
- 下载地址：https://www.dcloud.io/hbuilderx.html
- 选择"App开发版"
- 安装后直接打开项目即可运行

#### 方案B：命令行
```bash
# 1. 确保已安装 Node.js 16+
node -v

# 2. 进入项目目录
cd miniprogram

# 3. 安装依赖
npm install

# 4. 运行开发服务器
npm run dev:mp-weixin
```

### 2. 安装微信开发者工具
- 下载地址：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
- 下载后安装并使用微信扫码登录

## 🚀 快速启动（三种方式）

### 方式一：HBuilderX（最简单）⭐

1. 打开 HBuilderX
2. 文件 → 打开目录 → 选择 `miniprogram` 文件夹
3. 点击"运行"按钮 → "运行到小程序模拟器" → "微信开发者工具"
4. 微信开发者工具会自动打开并显示小程序

### 方式二：命令行

```bash
cd miniprogram
npm install
npm run dev:mp-weixin
```

然后在微信开发者工具中：
1. 点击"导入项目"
2. 选择 `miniprogram/dist/dev/mp-weixin` 目录
3. 点击"导入"

### 方式三：直接导入

1. 打开微信开发者工具
2. 点击"导入项目"
3. 选择 `miniprogram` 文件夹（如果使用命令行，选择编译后的目录）
4. AppID选择"测试号"或使用自己的AppID
5. 点击"导入"

## 📱 演示流程

### 演示顺序（建议）

1. **首页（30秒）**
   - 展示Banner轮播
   - 四大功能入口
   - 最新政策、推荐企业、热门课程

2. **政策查询（1分钟）**
   - 分类筛选功能
   - 政策列表展示
   - 点击进入政策详情
   - 展示政策完整信息

3. **企业画册（1分钟）**
   - 企业列表滚动
   - 搜索功能演示
   - 进入企业详情
   - 查看企业产品
   - 电话咨询功能（演示）

4. **企业课堂（30秒）**
   - 课程分类筛选
   - 课程列表展示
   - 课程详情查看

5. **我要求助（1分钟）**
   - 填写求助表单（演示）
   - 选择紧急程度
   - 提交成功提示
   - 跳转到我的求助

6. **个人中心（30秒）**
   - 用户信息展示
   - 功能菜单
   - 退出登录
