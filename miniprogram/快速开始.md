# 小程序Demo - 最简版本

## 🚀 超级简单的运行方法

### 方法一：使用HBuilderX（推荐）⭐

1. **下载安装HBuilderX**
   - 网址：https://www.dcloud.io/hbuilderx.html
   - 下载"App开发版"
   - 默认安装即可

2. **打开项目**
   - 启动HBuilderX
   - 文件 → 打开目录
   - 选择 `miniprogram` 文件夹

3. **运行项目**
   - 点击菜单"运行"
   - 选择"运行到小程序模拟器"
   - 选择"微信开发者工具"
   - （首次需要配置微信开发者工具路径）

4. **完成！**
   - 微信开发者工具会自动打开
   - 可以看到小程序界面

### 方法二：微信开发者工具直接导入

1. **准备项目**
   - 确保已安装Node.js
   - 在命令行执行：
     ```bash
     cd miniprogram
     npm install
     npm run dev:mp-weixin
     ```

2. **打开微信开发者工具**
   - 导入项目
   - 选择 `dist/dev/mp-weixin` 目录
   - AppID选择"测试号"

3. **完成！**
