# "当才当用"小程序Demo

这是"当才当用"政府服务平台的微信小程序演示版本，使用 uni-app 框架构建。

## ✨ 功能特性

✅ **完整的UI界面** - 所有核心功能模块的可视化界面
✅ **Mock数据** - 内置模拟数据，无需后端即可演示
✅ **精美设计** - 现代化UI设计，渐变色、动画效果
✅ **完整交互** - 所有页面跳转、表单提交等交互演示

## 📱 已实现功能

### 核心页面（9个）
1. **首页** - Banner轮播、功能入口、最新政策、推荐企业、热门课程
2. **政策查询** - 政策列表、分类筛选、政策详情
3. **企业课堂** - 课程列表、分类筛选、课程详情
4. **企业画册** - 企业列表、企业详情、产品展示
5. **我要求助** - 提交表单、我的求助记录
6. **我要采购** - 发布需求、我的需求记录
7. **个人中心** - 用户信息、功能菜单

## 🚀 快速开始

### 方式一：使用HBuilderX（推荐，最简单）

1. **下载HBuilderX**
   - 访问：https://www.dcloud.io/hbuilderx.html
   - 下载"App开发版"

2. **打开项目**
   - 启动 HBuilderX
   - 文件 → 打开目录 → 选择 `miniprogram` 文件夹

3. **运行小程序**
   - 点击工具栏的"运行"按钮
   - 选择"运行到小程序模拟器"→ "微信开发者工具"
   - 首次需要配置微信开发者工具路径

4. **微信开发者工具会自动打开并运行小程序**

### 方式二：使用命令行

```bash
# 1. 进入项目目录
cd miniprogram

# 2. 安装依赖
npm install

# 3. 运行开发服务器
npm run dev:mp-weixin

# 4. 使用微信开发者工具打开 dist/dev/mp-weixin 目录
```

## 🔧 常见问题

### Q: HBuilderX找不到微信开发者工具路径？
A: 工具 → 设置 → 运行配置 → 微信开发者工具路径，选择安装路径

### Q: 小程序无法预览？
A: 需要在微信开发者工具中登录微信账号

### Q: 页面跳转失败？
A: 检查pages.json中是否配置了对应页面路径
